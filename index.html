Japanese Writing

Let's Start from the Basics!

The Japanese writing system uses 3 script types - Hiragana, Katakana and Kanji. Have a look ➜

Hiragana

The foundation to be mastered! These are for commonly used native Japanese words.

Katakana

A level higher! This is used for foreign words, names, loanwords and/or scientific terms.

Kanji

Adopted from traditional Chinese, these are used to write content words and in formal conversation.
